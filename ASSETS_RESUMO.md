## 🎨 CHECKLIST DE CRIAÇÃO

### Antes de começar:
- [ ] Você tem fotos/referências para cada asset
- [ ] Você tem software (Canva, Figma, Photoshop)
- [ ] Você tem stock photos acessados (Unsplash, Pexels)
- [ ] Você definiu a paleta de cores (verde #6CB350)

### Durante criação:
- [ ] Imagens estão na dimensão correta
- [ ] Qualidade está boa (não pixelada)
- [ ] Cores combinam com marca
- [ ] Nenhuma marca d'água visível

### Antes de upload:
- [ ] Arquivo comprimido (<500KB)
- [ ] Nome descritivo
- [ ] Sem acentos/caracteres especiais
