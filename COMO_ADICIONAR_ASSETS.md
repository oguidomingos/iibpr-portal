# 🖼️ Passo a Passo — Como Adicionar Assets

Guia prático para adicionar images, banners e fotos no WordPress.

---

## 📍 Acessar WordPress Admin

1. Abra seu navegador
2. Va para: `https://seu-site.com/wp-admin`
3. Faça login com seu usuário
4. Você está no **Dashboard**

---

## 🎯 TAREFAS PRINCIPAIS

### 1️⃣ ADICIONAR LOGO

**Onde:** Aparência → Personalizar

#### Passo a passo:
```
1. No Dashboard, clique em "Aparência"
   └─ Veja o menu à esquerda

2. Clique em "Personalizar"
   └─ Abre o Customizer

3. No painel esquerdo, procure por "Identidade Visual"
   └─ Ou role até encontrar

4. Clique em "Identidade Visual"
   └─ Expand a seção

5. Procure por "Logo do Instituto"
   └─ Você verá um botão "Selecionar logo"

6. Clique em "Selecionar logo"
   └─ Abre a Biblioteca de Mídia

7. Clique em "Fazer upload"
   └─ Se logo não estiver já na biblioteca

8. Selecione sua imagem PNG (300×100 mínimo)
   └─ Pronto! Logo está adicionada

9. Clique em "Publicar" (topo direito)
   └─ Salva as mudanças no site ao vivo
```

#### Arquivo esperado:
- **Nome:** `logo-iibpr.png`
- **Dimensão:** 300×100px (ou maior)
- **Formato:** PNG com transparência
- **Peso:** Máx 50KB

---

### 2️⃣ ADICIONAR HERO BANNERS

**Onde:** Aparência → Personalizar → Hero — Carrossel Principal

#### Passo a passo:
```
1. Aparência → Personalizar

2. No painel esquerdo, procure por "Hero — Carrossel Principal"
   └─ Ou role até encontrar

3. Clique em "Hero — Carrossel Principal"
   └─ Expand

4. Você verá "Slide 1", "Slide 2", etc.
   └─ Clique no slide que quer editar

5. Dentro do slide, procure por "Imagem de Fundo"
   └─ Clique em "Selecionar imagem"

6. Na Biblioteca de Mídia:
   a) Se imagem já existe, selecione
   b) Se não existe, clique "Fazer upload"

7. Upload da imagem:
   a) Clique no botão de upload
   b) Selecione arquivo 1920×1080
   c) Aguarde upload completar
   d) Clique na imagem para selecionar
   e) Clique em "Selecionar" (topo)

8. Você voltará ao slide
   └─ Imagem agora aparece em "Imagem de Fundo"

9. Repita para Slide 2, 3, 4, 5

10. Clique em "Publicar" (topo direito)
    └─ Salva todas as mudanças
```

#### Arquivos esperados:
```
iibpr_hero_slide_1_image → 1920×1080 JPG (~400KB)
iibpr_hero_slide_2_image → 1920×1080 JPG (~400KB)
iibpr_hero_slide_3_image → 1920×1080 JPG (~400KB)
iibpr_hero_slide_4_image → 1920×1080 JPG (~400KB)
iibpr_hero_slide_5_image → 1920×1080 JPG (~400KB)
```

#### Dica:
Se você só quer 3 slides, deixe os campos de Slide 4 e 5 **vazios**. O site detecta automaticamente e só mostra os preenchidos.

### 3️⃣ ADICIONAR CAPAS DE CURSOS

**Onde:** Cursos → Selecionar Curso → Imagem Destacada

#### Passo a passo para CADA CURSO:
```
1. No Dashboard, clique em "Cursos"
   └─ Ou em menu principal à esquerda

2. Você vê lista de cursos
   └─ Clique no curso que quer editar

3. Abre o editor do curso
   └─ Role até "Imagem Destacada" (lado direito)
   └─ Você pode estar com a visão "Documento" ou "Editor"

4. Clique em "Definir imagem destacada"
   └─ Abre Biblioteca de Mídia

5. Se imagem não existe ainda:
   a) Clique "Fazer upload"
   b) Selecione seu arquivo 600×400
   c) Aguarde upload
   d) Clique na imagem quando aparecer
   e) Clique "Selecionar" (topo)

6. Se imagem já existe na biblioteca:
   a) Procure pela imagem
   b) Clique nela
   c) Clique "Selecionar" (topo)

7. Você voltará ao editor do curso
   └─ Imagem aparece agora como "Imagem Destacada"

8. Role até o topo e clique "Publicar" ou "Atualizar"
   └─ Salva mudanças do curso

9. Repita processo para CADA CURSO
   └─ Volta à lista de Cursos
   └─ Clique no próximo
```

#### Arquivo esperado por curso:
```
Especialização em Psicomotricidade
├─ Imagem: 600×400 JPG
├─ Nome: card-curso-especializacao.jpg
└─ Peso: ~150KB

Grafomotricidade
├─ Imagem: 600×400 JPG
├─ Nome: card-curso-grafomotricidade.jpg
└─ Peso: ~150KB

[Repita para cada curso...]
```

### 4️⃣ ADICIONAR CAPAS DE EVENTOS

**Onde:** Eventos → Selecionar Evento → Imagem Destacada

#### Passo a passo (IDÊNTICO aos cursos):
```

2. Clique no evento para editar

3. Role até "Imagem Destacada" (lado direito)

4. Clique "Definir imagem destacada"

5. Fazer upload ou selecionar existente:

   b) Ou procure na biblioteca existente

6. Clique "Selecionar"

7. Clique "Publicar" ou "Atualizar"

8. Repita para CADA EVENTO
```

#### Arquivo esperado por evento:
```

├─ Imagem: 600×400 JPG
├─ Nome: card-evento-seminario.jpg
└─ Peso: ~150KB

├─ Imagem: 600×400 JPG
├─ Nome: card-evento-palestra.jpg
```

## 📤 UPLOAD DA BIBLIOTECA DE MÍDIA

### Fazer upload de arquivo:

#### Método 1: Via Customizer
```
1. Aparência → Personalizar
2. Clique em qualquer campo que diga "Selecionar [imagem/logo]"
3. Abre Biblioteca de Mídia
4. Clique "Fazer upload"
5. Arraste arquivo ou clique para selecionar
6. Aguarde conclusão
7. Clique na imagem quando aparecer
8. Clique "Selecionar" (topo)
```

#### Método 2: Via Mídia (Dashboard)
```
1. Dashboard → Mídia
2. Clique "Adicionar novo"
3. Arraste arquivo aqui ou clique para selecionar
4. Aguarde upload
5. Você pode editar título, alt text, descrição
6. Clique "Publicar"
```

## ✅ CHECKLIST DE UPLOAD

Antes de fazer upload, marque:

- [ ] Arquivo está na dimensão correta
- [ ] Arquivo está em formato correto (JPG ou PNG)
- [ ] Arquivo tem nome descritivo
- [ ] Arquivo foi comprimido (máx tamanho especificado)
- [ ] Arquivo não tem caracteres especiais no nome
- [ ] Arquivo é legível (boa qualidade)

### Se cometi um erro:

#### Trocar imagem:
```
1. Va ao local onde está a imagem
   (Customizer ou CPT)

2. Clique na imagem atual

3. Clique em "Remover" ou
   "Selecionar outra imagem"

4. Upload a nova

5. Clique "Publicar"
```

#### Deletar imagem:
```
1. Dashboard → Mídia

2. Procure a imagem

3. Passe mouse sobre ela

4. Clique em "Deletar permanentemente"

5. Confirme
```

## 🎥 PREVIEW EM TEMPO REAL

### Visualizar mudanças enquanto edita:

#### No Customizer:
```
1. Aparência → Personalizar

2. À direita você vê o PREVIEW do site

3. Enquanto edita, mudanças aparecem em tempo real

4. Se não aparecer preview:
   a) Clique em "Área do Site" (topo esquerdo)
   b) Selecione página para preview
```

#### Testar na página real:
```
1. Depois de "Publicar" mudanças

2. Va para página frontend:
   https://seu-site.com

3. Faça Ctrl+F5 (limpar cache)

4. Verifique se imagem aparece corretamente
```

## 🔧 TROUBLESHOOTING

### Problema: Imagem não aparece

**Causa 1:** Imagem ainda está enviando
- Aguarde upload completar (veja barra de progresso)

**Causa 2:** Cache do navegador
- Faça Ctrl+F5 ou limpe cache
- Tente outro navegador

**Causa 3:** Imagem muito pesada
- Comprima arquivo com TinyPNG
- Reduza dimensões se necessário

**Causa 4:** Formato errado
- Verifique se é JPG ou PNG
- Tente salvar em outro formato

### Problema: Imagem aparece distorcida

**Solução:**
- Imagem pode estar em proporção errada
- Verifique dimensões (ex: deve ser 600×400, não 800×600)
- Reajuste imagem antes de upload

### Problema: Imagem não cabe na área

**Solução:**
- Imagem está grande demais
- Comprima mantendo proporção correta
- Se necessário, recorte para ajustar

### Problema: Texto não legível sobre imagem

**Solução:**
- Imagem muito clara/brilhante
- Customize aumenta overlay escuro
- Ou escolha imagem com mais contraste

## 🎓 Exemplo Completo: Adicionar um Curso Novo

### Cenário: Você quer adicionar um novo curso "Psicomotricidade Avançada"

```
Passo 1: Criar o curso
├─ Dashboard → Cursos
├─ Clique "Adicionar novo"
├─ Preencha título: "Psicomotricidade Avançada"
└─ Preencha outros campos (descrição, preço, etc)

Passo 2: Adicionar imagem destacada
├─ Procure "Imagem Destacada" (lado direito)
├─ Clique "Definir imagem destacada"
├─ Fazer upload de: card-curso-avancada.jpg (600×400)
└─ Clique "Selecionar"

Passo 3: Publicar
├─ Clique "Publicar" (topo)
└─ Curso agora aparece no site com capa de imagem

Passo 4: Verificar
├─ Va para /cursos no front-end
├─ Faça Ctrl+F5 (limpar cache)
├─ Verifique se curso aparece com imagem
└─ Pronto! ✅
```
